# Tokens

- **`rantai-lake-theme.css`** — variabel `:root`, `.dark`, dan blok `@theme inline` untuk Tailwind v4 + shadcn. Sisipkan ke `globals.css` proyek baru setelah impor Tailwind / shadcn (lihat komentar di bagian atas file).

Sumber kebenaran di aplikasi: `src/app/globals.css`. Jika Anda mengubah tema di app, sinkronkan salinan di folder ini.
